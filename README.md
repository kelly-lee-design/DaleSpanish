# ¡DALE! 🇪🇸

> **한국인을 위한 스페인어 학습 웹 앱**

## 📖 ¡DALE! 이란?

**¡DALE!** 은 한국어 화자를 위해 설계된 스페인어 학습 웹 서비스입니다.
기존 언어 앱들이 영어 기반으로 되어 있어 한국인 학습자에게 불편했던 점을 해결하고자,
**한국어 UI · 한국어 뜻풀이 · 한국어 예문 해설**을 기본으로 제공합니다.

### 왜 ¡DALE!인가요?
- 🇰🇷 **100% 한국어 인터페이스** — 번역 없이 바로 이해
- 📱 **모바일 퍼스트** — 출퇴근길, 자투리 시간에 학습
- 🎯 **스페인어 실전 중심** — 실생활에서 자주 쓰는 단어·동사 우선
- 🔥 **습관 형성 시스템** — 스트릭과 체크인으로 꾸준함 유도
- 🤖 **AI 스마트 입력** — 단어/문장을 붙여넣으면 자동 분류·저장

### 핵심 기능
| 기능 | 설명 |
|------|------|
| 📚 단어 라이브러리 | 3D 플립 플래시카드 + 스와이프 네비게이션 |
| 🔤 필수 동사 100 | 시제별 예문과 함께 학습하는 핵심 동사 |
| ✍️ 스마트 입력 | AI 기반 단어/문장 자동 분류 저장 |
| 🎯 퀴즈 모드 | 복습 문제로 학습 내용 점검 |
| 🔥 스트릭 | 매일 체크인으로 연속 학습일 기록 |
| 📊 대시보드 | 학습 진행도와 통계를 한눈에 |
